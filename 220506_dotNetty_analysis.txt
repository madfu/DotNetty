DotNetty is an EventDriven high-performance asynchronous network framework, Netty, which is widely used in the Java camp.
Ported to C# .NET.
To use DotNetty in Unity GameEngine, we modified some code from the DotNetty repository.

DotNetty Repository -> https://github.com/Azure/DotNetty



Tested on Unity 2019.2.10f.
(Test Settings -> IL2CPP Build, Release Mode, API Compatibility .NET Standard 2.0 & .NET 4.x)

It has been confirmed that TCP and UDP communication works in Android, iOS, and Standalone Windows & MacOS builds.


How to use

You can use it after copying the DotNetty-0.6.0_forUnity Plugins folder to the Assets/Plugins folder of your Unity project.
